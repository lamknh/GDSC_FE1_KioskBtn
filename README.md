# GDSC_FE1_KioskBtn
GDSC FE1 스터디 과제  키오스크 버튼만들기

### 준수사항
```
- 제출하기 버튼이 눌렸을 때, 입력한 전화번호를 출력하는 페이지로 이동할것
- 해당 페이지의 디자인은 뭐 아무렇게나 해도 무방하나, 반드시 입력한 정보를 출력해야 함
- Container-Presenter 패턴을 적용할 것
  (즉 Presenter에 해당하는 것은 렌더링 이외의 것은 하지 않게 해야함)

- (선택) mui를 사용해서 만들어 보기
```
